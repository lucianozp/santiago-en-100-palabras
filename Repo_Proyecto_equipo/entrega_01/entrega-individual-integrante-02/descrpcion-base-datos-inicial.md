# Base de datos: Ganadores de Santiago en 100 Palabras (base construida)

Esta base de datos es de elaboración propia. Los datos los estamos construyendo a partir de los cuentos de Santiago en 100 palabras disponibles en la página web, donde publican los cuentos ganadores de cada edición. 

Los datos originales provienen de publicaciones oficiales del concurso (listados de ganadores por año), y han sido recopilados, sistematizados y estructurados.

## Contenido

La base de datos tiene información sobre los cuentos ganadores de las 12 ediciones del concurso (12-24). Se trata de datos estructurados que variables categóricas.

Variables consideradas:

Edición del cuento
Nombre del autor/a
Edad del autor/a
Comuna de origen del autor/a
Género del autor/a

Periodo de cobertura:

Desde la edición 12 hasta la edición más reciente disponible (12 últimas ediciones del concurso)

Pertinencia

Esta base de datos es demasiado importante para desarrollar nuestro proyecto, ya que es la primera parte de la investigación que permite identificar quiénes son los autores seleccionados y desde que comuna provienen. A partir de esta información, se pueden establecer relaciones entre el origen geográfico de los ganadores y variables socioeconómicas externas.

Esta base permitirá:

Analizar la distribución de comunas de los ganadores
ver la concenytración de comunas de los ganadores
Cruzar las variables como ingreso, educación y acceso cultural

De este modo, es la base principal para evaluar la hipótesis sobre desigualdad en el acceso y éxito dentro del concurso.

Metodología

Puesto que no existe una base de datos con esta información, estamos haciendo una propia base mediante un proceso de recolección y sistematización.

Método de recolección:

Revisión de forma manual las ediciones, identificando el nombre del autor/a, su comuna y edad.
Registro en un excel

Proceso:

Identificación de los cuentos ganadores en las últimas 12 ediciones
Registro del nombre del autor/a
Edad del autor/a
Comuna del autor/a
Género del autor/a

La única limitación es que una vez terminada esta base de datos, dependemos de otras fuentes.
