# Base de datos: Ganadores de Santiago en 100 Palabras (base construida)

Esta base de datos es de elaboración propia. Los datos los estamos construyendo a partir de los cuentos de Santiago en 100 palabras disponibles en la página web, donde publican los cuentos ganadores de cada edición. 

Los datos originales provienen de publicaciones oficiales del concurso (listados de ganadores por año), y han sido recopilados, sistematizados y estructurados.

## Contenido

La base de datos contiene información sobre los cuentos ganadores de las 24 ediciones del concurso. Se trata de datos estructurados en formato tabular (CSV/Excel), que incluyen variables categóricas.

Variables consideradas:

Edición del cuento
Nombre del autor/a
Edad del autor/a
Comuna de origen del autor/a
Género del autor/a

Periodo de cobertura:

Desde 2001 hasta la edición más reciente disponible (24 ediciones del concurso)

Pertinencia

Esta base de datos es demasiado importante para desarrollar nuestro proyecto, ya que es la primera parte de la investigación que permite identificar quiénes son los autores seleccionados y desde que comuna provienen. A partir de esta información, se pueden establecer relaciones entre el origen geográfico de los ganadores y variables socioeconómicas externas.

En particular, esta base permitirá:

Analizar la distribución territorial de los ganadores
Detectar posibles concentraciones en ciertas comunas
Cruzar los datos con indicadores como ingreso, educación y acceso cultural

De este modo, es la base principal para evaluar la hipótesis sobre desigualdad en el acceso y éxito dentro del concurso.

Metodología

Puesto que no existe una base de datos con esta información, harémos una propia base mediante un proceso de recolección y sistematización.

Método de recolección:

Revisión manual de los sitios oficiales del concurso
Extracción de información desde listados de ganadores por año
Registro manual en planilla estructurada

Proceso:

Identificación de los cuentos ganadores por cada edición
Registro del nombre del autor/a
Búsqueda complementaria (cuando es posible) para obtener:
Edad del autor/a
Comuna del autor/a
Género del autor/a

Limitaciones:

Dependencia de fuentes secundarias para completar variables
